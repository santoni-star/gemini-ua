# Розширення Gemini CLI

Розширення Gemini CLI дозволяють об'єднувати підказки, сервери MCP та власні
команди у зручний формат. За допомогою розширень ви можете розширювати
можливості Gemini CLI та ділитися ними з іншими.

## Керування розширеннями

Керування здійснюється через команди `gemini-ua extensions` у терміналі. Ці
зміни набудуть чинності після перезапуску CLI.

### Встановлення розширення

Ви можете встановити розширення за посиланням на GitHub або через локальний
шлях.

```bash
gemini-ua extensions install <джерело>
```

### Видалення розширення

```bash
gemini-ua extensions uninstall <назва>
```

### Вимкнення розширення

Ви можете вимкнути розширення глобально або для конкретного робочого простору.

```bash
gemini-ua extensions disable <назва> [--scope <user|workspace>]
```

### Оновлення розширення

```bash
gemini-ua extensions update <назва>
# Або всі відразу:
gemini-ua extensions update --all
```

## Як це працює

Gemini CLI шукає розширення у папці `~/.gemini/extensions/`. Кожне розширення —
це папка, яка обов'язково містить файл `gemini-extension.json`.

### Файл `gemini-extension.json`

Приклад структури:

```json
{
  "name": "my-extension",
  "version": "1.0.0",
  "mcpServers": {
    "my-server": {
      "command": "node my-server.js"
    }
  },
  "contextFileName": "GEMINI.md",
  "excludeTools": ["run_shell_command"]
}
```

- `name`: Унікальна назва розширення (маленькими літерами через дефіс).
- `version`: Версія.
- `mcpServers`: Список серверів MCP, які будуть запущені разом із розширенням.
- `contextFileName`: Файл контексту для розширення.
- `excludeTools`: Список інструментів, які заборонено використовувати цьому
  розширенню.

## Власні команди в розширеннях

Розширення можуть надавати власні команди. Для цього створіть папку `commands/`
всередині папки розширення та додайте туди `.toml` файли.

**Вирішення конфліктів:** Команди розширень мають найнижчий пріоритет. Якщо
назва збігається з вашою власною командою, команда розширення отримає префікс
(наприклад, `/gcp.deploy` замість `/deploy`).
