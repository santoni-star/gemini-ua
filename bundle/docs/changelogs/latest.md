# Останній стабільний реліз: v0.21.0 - v0.21.1

Дата виходу: 16 грудня 2025 року

Для більшості користувачів ми рекомендуємо використовувати саме цю версію.
Встановити останній стабільний реліз можна командою:

```
npm install -g @google/gemini-cli
```

## Ключові зміни

- **⚡️⚡️⚡️ Gemini 3 Flash + Gemini CLI:** Платні користувачі тепер можуть
  увімкнути Gemini 3 Pro та Gemini 3 Flash. Перейдіть у `/settings` та
  встановіть **Preview Features** у значення `true`. Докладніше:
  [Gemini 3 Flash тепер доступна у Gemini CLI](https://developers.googleblog.com/gemini-3-flash-is-now-available-in-gemini-cli/).

## Що змінилося

- Рефакторинг `stdio`: покращено обробку виводу для усунення артефактів
  рендерингу.
- Впроваджено нечіткий пошук усередині діалогу налаштувань.
- Увімкнено інтеграцію з шиною повідомлень за замовчуванням.
- Додано рекомендацію використовувати `--debug` замість `--verbose` для
  налагодження CLI.
- Консолідовано віддалені сервери MCP для використання поля `url` у
  конфігурації.
- Обмежено інструменти, доступні в інтеграційних тестах.
- Додано відстеження назв репозиторіїв GitHub у подіях телеметрії.
- Дозволено експортерам телеметрії GCP використовувати облікові дані входу
  користувача.
- Виправлено вразливості безпеки NPM у пакунках `glob` та `body-parser`.
- Додано нові інструкції для корпоративних користувачів.
- Впроваджено хуки для життєвого циклу сесії та стиснення контексту.
- Додано панель команд для хуків (увімкнення/вимкнення/міграція).
- Додано підтримку хуків розширень із попередженням про безпеку.
- Додано експериментальний прапорець `enableAgents`.
- Виправлено навігацію по словах для символів CJK.
- Додано команди для перегляду та оновлення налаштувань конкретних розширень.
- Додано підтримку `/copy` у віддалених сесіях через OSC52.
- Покращено логіку повторних спроб для мережевих помилок `fetch`.
- Додано підтримку динамічного оновлення інструментів MCP.
- Виправлено екран конфіденційності для користувачів застарілих тарифних планів.
- Покращено обробку зображень у буфері обміну (підтримка форматів Gemini API).
- Додано команду `restore` для сервера A2A.
