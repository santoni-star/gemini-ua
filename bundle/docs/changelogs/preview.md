# Попередня версія (Preview): Реліз v0.22.0-preview.0

Дата виходу: 16 грудня 2025 року

Наш канал попередніх релізів містить найновіші та експериментальні функції. Ця
версія може бути менш стабільною, ніж наш
[останній щотижневий реліз](latest.md).

Щоб встановити попередню версію:

```
npm install -g @google/gemini-cli@preview
```

## Що змінилося

- Впроваджено підтримку змінної середовища `GEMINI_CLI_IDE_AUTH_TOKEN` як
  резервної.
- Додано відображення статистики квот для невикористаних моделей у `/stats`.
- Codebase investigator тепер використовує preview-модель синхронно з основним
  агентом.
- Додано причину закриття у робочий процес застарілих багів.
- Реалізовано передачу версії моделі та CLI у заголовку User Agent.
- Рефакторинг: генерація підсумку сесії перенесена на етап запуску.
- Обмежено глибину пошуку в коректорі шляхів.
- Впроваджено механізм відкладеного вирішення конфігурацій моделей.
- Додано підтримку `transcript_path` у хуках для розширення git-ai/Gemini.
- Рефакторинг: DelegateToAgentTool реалізовано через discriminated union.
- Додано підтримку перетягування (drag and drop) для кількох зображень.
- Покращено валідацію схем налаштувань.
- Додано команду `/init` для сервера A2A.
- Виправлено дефолтні політики для codebase_investigator (дозволено в
  read-only).
- Покращено генерацію ліцензій для залежностей розширення VS Code.
- Впроваджено постійні політики "Always Allow" для shell та MCP.
- Додано подію телеметрії `gemini_cli.startup_stats`.
- Оновлено документацію по двигуну політик (policy engine).
- Виправлено фрагментацію виводу інструментів.
- Реалізовано відображення сирої кількості вхідних токенів у JSON виводі.
- Виправлено пріоритетність злиття налаштувань (новий формат JSON тепер має
  пріоритет).
