# Gemini CLI: Квоти та ціни

Gemini CLI пропонує щедрий безкоштовний рівень, який покриває більшість потреб
індивідуальних розробників. Для корпоративного використання або якщо вам
потрібні вищі ліміти, доступно кілька варіантів залежно від вашого методу
аутентифікації.

Деталі щодо політики конфіденційності та умов використання дивіться у розділі
[Конфіденційність та умови](./tos-privacy.md).

Цей документ описує квоти та ціни для різних методів входу.

Загалом є три категорії:

- **Безкоштовне використання:** Ідеально для експериментів.
- **Платні плани (фіксована ціна):** Для тих, кому потрібні вищі ліміти та
  передбачувані витрати.
- **Оплата за фактом використання (Pay-As-You-Go):** Найбільш гнучкий варіант
  для професійної роботи та тривалих завдань.

## Безкоштовне використання

Ваші ліміти залежать від методу аутентифікації:

### Вхід через Google (Gemini Code Assist для фізичних осіб)

- 1000 запитів до моделі / день
- 60 запитів / хвилину
- Використовує різні моделі родини Gemini за вибором CLI.

### Вхід через Gemini API Key (без оплати)

- 250 запитів до моделі / день
- 10 запитів / хвилину
- Тільки модель Flash.

### Вхід через Vertex AI (Express Mode)

- 90 днів безкоштовного використання без увімкнення білінгу.
- Квоти та моделі залежать від типу акаунта.

## Платні плани: Вищі ліміти за фіксовану ціну

Якщо безкоштовних запитів недостатньо, ви можете перейти на платну підписку:

- **Google AI Pro та AI Ultra:** Рекомендовано для індивідуальних розробників.
- **Gemini Code Assist для організацій (через Google Cloud):**
  - **Standard:** 1500 запитів / день, 120 запитів / хв.
  - **Enterprise:** 2000 запитів / день, 120 запитів / хв.

## Оплата за фактом (Pay As You Go)

Якщо ви вичерпали всі квоти, перейдіть на модель оплати за кожен запит/токен.
Для цього використовуйте Gemini API Key або Vertex AI (Regular Mode).

- **Vertex AI:** Квоти регулюються динамічно, оплата базується на використанні
  токенів.
- **Gemini API key:** Ціна залежить від моделі та обсягу токенів.

**Важливо:** При використанні API-ключа ви платите за кожен токен. Це може бути
дорожче при великій кількості дрібних запитів, але це єдиний спосіб гарантувати
роботу без обмежень квоти.

## Поради для економії

- Не приймайте кожну пропозицію ШІ наосліп, особливо для складних завдань типу
  рефакторингу великих кодових баз.
- Будьте конкретними у підказках. Оскільки ви платите за кожен виклик, думайте
  про найефективніший спосіб досягнення результату.

## Gemini API проти Vertex AI

- **Gemini API:** Найшвидший спосіб використання моделей напряму.
- **Vertex AI:** Корпоративна платформа з підвищеними вимогами до безпеки та
  контролю.

## Перегляд статистики

Підсумок використання доступний через команду `/stats` та виводиться в кінці
кожної сесії.
