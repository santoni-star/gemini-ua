# Інструмент веб-пошуку (`google_web_search`)

Цей документ описує інструмент `google_web_search`.

## Опис

Використовуйте `google_web_search` для виконання пошуку в Інтернеті через Google
Search безпосередньо за допомогою Gemini API. Інструмент повертає підсумок
знайденої інформації з посиланнями на джерела.

### Аргументи

- `query` (рядок, обов'язково): Пошуковий запит.

## Як працює `google_web_search`

Інструмент надсилає запит до Gemini API, який виконує веб-пошук. Gemini повертає
згенеровану відповідь на основі результатів пошуку, включаючи цитати та
посилання на джерела.

**Важливо:** Інструмент повертає оброблений підсумок, а не просто список
посилань.

## Приклади

**Отримання інформації на певну тему:**

```text
google_web_search(query="останні досягнення у генерації коду за допомогою ШІ")
```
