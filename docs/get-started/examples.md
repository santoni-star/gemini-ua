# Приклади використання Gemini CLI

Не знаєте з чого почати? Цей документ містить приклади використання Gemini CLI
для різних завдань.

**Примітка:** Наведені результати є прикладами. Ваші результати можуть
відрізнятися.

## Перейменування фотографій на основі їхнього вмісту

Сценарій: У вас є папка з такими файлами:

```bash
photos/photo1.png
photos/photo2.png
photos/photo3.png
```

Дайте Gemini таку підказку:

```cli
Rename the photos in my "photos" directory based on their contents.
```

Результат: Gemini попросить дозволу на перейменування ваших файлів.

Виберіть **Allow once**, і ваші файли будуть перейменовані:

```bash
photos/жовті_квіти.png
photos/антикварний_комод.png
photos/зелений_робот_андроїд.png
```

## Огляд репозиторію шляхом читання його коду

Сценарій: Ви хочете зрозуміти, як працює популярна утиліта з відкритим кодом,
вивчивши її код, а не лише README.

Дайте Gemini CLI таку підказку:

```cli
Clone the 'chalk' repository from https://github.com/chalk/chalk, read its key source files, and explain how it works.
```

Результат: Gemini виконає послідовність дій для відповіді на ваш запит.

1.  Спочатку він попросить дозвіл на запуск `git clone`.
2.  Далі він знайде важливі файли вихідного коду та попросить дозвіл на їх
    читання.
3.  Нарешті, після аналізу коду, він надасть підсумок.

Gemini CLI поверне пояснення на основі реального коду:

```markdown
Бібліотека `chalk` — це популярний npm-пакет для стилізації виводу в терміналі
за допомогою кольорів. Після аналізу вихідного коду, ось як вона працює:

- **Основна функціональність:** Головний файл налаштовує ланцюговий API. Кожен
  колір або модифікатор (наприклад, `bold` або `italic`) — це гетер, який додає
  відповідний керуючий код ANSI до внутрішнього стеку.
- **Ланцюговий API:** Коли ви викликаєте `chalk.red.bold('Привіт')`, ви
  звертаєтесь до гетерів.
- **Генерація виводу:** Коли ланцюжок використовується як рядок, викликається
  метод `toString()`, який обгортає вхідний рядок усіма збереженими кодами ANSI
  та додає код скидання в кінці.
```

## Об'єднання двох електронних таблиць в одну

Сценарій: У вас є два файли .csv: `Revenue - 2023.csv` та `Revenue - 2024.csv`.
Кожен файл містить щомісячні показники доходу. Ви хочете об'єднати їх в один
файл.

Дайте Gemini CLI таку підказку:

```cli
Combine the two .csv files into a single .csv file, with each year a different column.
```

Результат: Gemini CLI прочитає кожен файл, а потім попросить дозволу на запис
нового файлу. Після вашої згоди він створить об'єднаний файл .csv.

## Написання юніт-тестів

Сценарій: Ви написали просту сторінку входу. Ви хочете написати юніт-тести, щоб
забезпечити покриття коду.

Дайте Gemini CLI таку підказку:

```cli
Write unit tests for Login.js.
```

Результат: Gemini CLI попросить дозволу на запис нового файлу та створить тест
для вашої сторінки входу.
