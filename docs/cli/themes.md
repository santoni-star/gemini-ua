# Теми оформлення

Gemini CLI підтримує різноманітні теми для налаштування колірної схеми та
зовнішнього вигляду. Ви можете змінити тему відповідно до ваших уподобань за
допомогою команди `/theme` або параметра `"theme"` у конфігурації.

## Доступні теми

Gemini CLI постачається з набором попередньо визначених тем, список яких можна
переглянути командою `/theme`:

- **Темні теми:**
  - `ANSI`
  - `Atom One`
  - `Ayu`
  - `Default`
  - `Dracula`
  - `GitHub`
- **Світлі теми:**
  - `ANSI Light`
  - `Ayu Light`
  - `Default Light`
  - `GitHub Light`
  - `Google Code`
  - `Xcode`

### Як змінити тему

1.  Введіть `/theme` у Gemini CLI.
2.  З'явиться діалогове вікно зі списком доступних тем.
3.  Використовуйте клавіші зі стрілками для вибору. Деякі теми підтримують
    попередній перегляд під час вибору.
4.  Підтвердьте вибір (`Enter`), щоб застосувати тему.

**Примітка:** Якщо тему жорстко вказано у файлі `settings.json`, ви повинні
видалити рядок `"theme"`, щоб мати змогу змінювати її через команду `/theme`.

## Власні кольорові теми

Gemini CLI дозволяє створювати власні теми, вказуючи їх у файлі `settings.json`.

### Як визначити власну тему

Додайте блок `customThemes` до вашого файлу `settings.json`. Кожна тема — це
об'єкт з унікальною назвою та набором ключів кольорів.

```json
{
  "ui": {
    "customThemes": {
      "MyCustomTheme": {
        "name": "MyCustomTheme",
        "type": "custom",
        "Background": "#181818",
        "Foreground": "#F8F8F2",
        "LightBlue": "#82AAFF",
        "AccentBlue": "#61AFEF",
        "AccentPurple": "#BD93F9",
        "AccentCyan": "#8BE9FD",
        "AccentGreen": "#50FA7B",
        "AccentYellow": "#F1FA8C",
        "AccentRed": "#FF5555",
        "Comment": "#6272A4",
        "Gray": "#ABB2BF"
      }
    }
  }
}
```

Ви можете використовувати HEX-коди (наприклад, `#FF0000`) або стандартні назви
кольорів CSS (`coral`, `teal`, `blue`).

### Завантаження теми з файлу

Ви також можете завантажити тему прямо з файлу JSON:

```json
{
  "ui": {
    "theme": "/шлях/до/вашої/теми.json"
  }
}
```

**Примітка з безпеки:** Для вашої безпеки Gemini CLI завантажує файли тем лише з
вашого домашнього каталогу.
